Html

  1. Name the components of html elements properly

    2. The components are a text header using <hx>, a paragraph using <p>, a horizontal ruler using <hr>, a link using <a>, a list using <ul> or <ol gt and <li> for listing its list elements, an image using <img>, a divider using <div> and a text span using <span>

  2. Identify void elements and how they are denoted

    3. They only have a start tag since they can't have any content

  3. Identify attributes

    4. Entities contain attributes, which are characteristics or modifiers, qualities, amounts, or features

  4. Write down the correct DOCTYPE declaration for html 5

    5. <!DOCTYPE>

  5. Write down syntactically correct statements for the following elements
    • Article (<article>): <article><h1>Hello World</h1></article>
    • Body (<body>): <body><h1>Hello World</h1></body>
    • line break (<br/>): <p>This<br>is a line break</p>
    • Headings (<h1>, ...): <h1>This is a headline</h1>
    • Section with meta-information (<head>): <head><title>This is a title</title></head>
    • Top level element (<html>): <html><body></body></html>
    • Image named Team.jpg (<img src="Team.jpg">): <img src="image.png">
    • Paragraph (<p>): <p>This is a paragraph</p>
    • Section (<section>): <section><h1>This is a header</h1></section>
    • Label appearing in the title bar (<title>): <title>This is a title</title>
    • Link to other pages (<a>): <a href="https://www.google.at">
    • Comment: <!-- This is a comment -->
    • Head with title and meta elements: <head><title>This is a title</title><style></style></head>

Css

  1. Name the components of a css rule properly

    2. <style> h1:{ style properties } </style>

  2. Use combinators properly: direct child, descendant, adjacent sibling, sibling

    3. selector1 > selector2 { style properties }
    selector1 selector2 {/* property declarations */}
    last_Element + Targetelement { style properties }
    element ~ element { style properties }

  3. Specify colors by color names and in hexadecimal notation

    4. color: blue;
    color: rgb(0, 0, 0);

  4. Declarations for color, background color

    5. color: white;
    background-color: white;

  5. Pseudo classes for the <a> element

    6. a:hover
    a:link
    a:visited
    a:active

  6. Declaration for background image

    7. background-image("image.png");

  7. Declaration for text alignment (left, right, center, justify)

    8. text-align: left;
    text-align: center;
    text-align: right;
    text-align: justify:

  8. Declaration for text decoration (overline, underline, line-through)

    9. text-decoration: underline;
    text-decoration: overline;
    text-decoration: line-through;

  9. Declaration for text transformation (uppercase, lowercase, capitalize)

    10. text-transform: uppercase;
    text-transform: lowercase;
    text-transform: capitalize;

  10. Identify the difference between serif and sans-serif fonts

    11. Serif = a small line
    Sans-serif = a type character without the above

  11. Declarations for font families

    12. font-family: Verdana;

  12. Declarations for font style normal and italic

    13. font-style: normal;
    font-style: italic;

  13. Declarations for font sizes

    14. font-size: 12px;

  14. Declarations for font weights

    15. font-weight: normal;
    font-weight: bold;
    font-weight: lighter;
    font-weight: bolder
